# 🚦 AI Traffic Management System

##  Overview
This project is a simulation of an AI-based traffic control system that dynamically manages traffic signals based on vehicle density.

 Features
- 4-way traffic intersection (North, South, East, West)
- Real-time vehicle simulation
- Adaptive signal timing
- Least traffic lane suggestion
- Interactive dashboard UI

 Working
The system calculates the number of vehicles in each lane and:
- Assigns green signal to the lane with highest traffic
- Suggests the least crowded lane
- Simulates vehicle movement

 Technologies Used
- HTML
- CSS
- JavaScript

 Future Scope
- Integration with AI (YOLO for vehicle detection)
- Real-time traffic camera input
- Smart city implementation
